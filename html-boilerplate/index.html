Indenting elements wtihin other elements is called nesting. When we nest elements we create a parent/child relationship between them. The nested elements are the children. We have used nesting so far to create this web page.

Comments are not visible in the browser, and are used to make future additions easier for other developers or our future selves. It gives us a point to jump off from with what we were trying to accomplish.

Anchor elements are how we create links in HTML. Anchor elements create the text of the link, and have a reference to where the link will take you.

click me

Generally, there are two kinds of links we will create:

  1. Links ot pages on other websites on the internet
  2. Links to pages located on our own websites

Aboslute links are links to pages on other websites on the internet. A typical absolute link has the following syntax: [protocol]://[domain]/[path]. An absolute link will always contain the protocol and domain of the destination. An example of an absolute link is the one we just created to the Odin Project website.

Relative link are links to other pages within our own website. These links do not include the domain name since it is another page in the same site. Relative links only include the file path to the other page relative to the page you are creating the link on. An example will be shown below.

Can you imagine a website with no images? Certainly not. HTML gives us several elements to display various kinds of media. the img tag is the most widely used of these. The img is self closing which means it does not need a closing tag. There is an attribute called src which tells the browser where the image file is located. Src works very similar to href. Here is an example using an absolute link to display Odin from the Odin Project!

If we wanted to use an image in the about pagee that we have stored locally we would create a path to the parent directory of our pages and then to the images directory, finally we can point to the image we want to use.

Every image should have an src attribute, but it should also have an alt attribute. The alt attribute is used to describe an image in case it cannot be loaded. It may also be used with screen readers to describe the image to visually impaired persons.
